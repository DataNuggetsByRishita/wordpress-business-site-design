# WordPress Business Website Project

## Industry Focus
Digital Business / Web Presence / Content Management

## Project Overview
This project involved designing and developing a professional business website using WordPress. The objective was to create a clear, user-friendly site structure that effectively communicates organisational information, supports content management needs, and reflects professional design standards.

The project demonstrates practical skills in website setup, content organisation, and user-focused design rather than purely technical development.

---

## Project Objectives
- Design a professional and intuitive website layout using WordPress
- Create and manage structured content suitable for a business or organisational context
- Apply usability and accessibility principles to improve user experience
- Demonstrate understanding of digital presence and online communication for organisations

---

## Tools & Skills Used
- WordPress (content management system)
- Website structure and navigation design
- Content creation and organisation
- User experience (UX) and usability principles
- Digital communication and presentation
- Basic HTML awareness (page structure)

---

## Key Features
- Clear navigation and page hierarchy
- Consistent layout and visual presentation
- Structured content aligned with business communication goals
- Responsive design suitable for different screen sizes
